# MetricDrive v0.1.0

MetricDrive v0.1.0 is the first public portfolio baseline. It packages a
complete laptop-scale planning-alignment loop: scenarios, trajectory metrics,
preference generation, learned reward modeling, hard-negative stress tests,
VLM-style planning rows, a tiny metric-reward policy optimization analogue, and
a static explorer.

Public baseline for a metric-aligned autonomous-driving planning portfolio
artifact.

The release is intentionally public-safe and independent. It uses controlled
synthetic scenarios and transparent metrics rather than private company data.

## Highlights

- Synthetic long-tail scenario core.
- Planning metrics and SVG trajectory visualization.
- Planner baselines: token/imitation proxy, progress-only, metric rerank.
- Metric-derived preference pairs.
- Learned Bradley-Terry reward model.
- Objective ablations.
- Hard-negative trajectory generation and stress evaluation.
- VLM-style planning examples with verifiable meta-actions.
- Tiny metric-reward policy optimization analogue.
- Static GitHub Pages demo export.
- README, portfolio report, recruiting packet, citation, CI, and metadata docs.

## Validation

```bash
python3 -m unittest discover
metricdrive score
metricdrive preferences
metricdrive learned --epochs 80
metricdrive ablations --epochs 80
metricdrive hard-negatives --epochs 20
metricdrive vlm-examples --limit 3
metricdrive rl-align --epochs 10
metricdrive export-demo --output docs/demo --epochs 20
git diff --check
```
